Place your SN Pro webfont files here (preferably .woff2).

Recommended filenames (used by the @font-face rules in `globals.css`):
- SNPro-Regular.woff2
- SNPro-Medium.woff2
- SNPro-Semibold.woff2
- SNPro-Bold.woff2

After adding the files, re-run the dev server or `npm run build` to ensure fonts are loaded.

If your font files use different names, update the `src` paths in the @font-face declarations inside `app/globals.css` accordingly.
